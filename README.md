# ERSHAD2 Web Application

A modern web-based academic advising system for CCIT students. Analyzes transcripts and provides course registration recommendations.

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Features

- PDF transcript upload & parsing
- Smart course recommendations
- Prerequisite checking
- Progress tracking for electives
- Downloadable reports

## Technology

- Next.js 14+ (TypeScript)
- TailwindCSS
- pdf-parse
